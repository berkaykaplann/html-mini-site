HTML (Hypertext Markup Language) web sayfalarını oluşturmak için kullanılan bir işaret dili veya kodlama dilidir. Web tarayıcısı tarafından yorumlanarak web sayfalarını düzenler ve gösterir. HTML, web sayfalarındaki metin, görüntü, video, bağlantı ve diğer öğelerin düzenlenmesine yardımcı olan birçok etiket içerir.

Kısaca HTML site kodu olarak kullanılır. Yani bir web sitesi yapmaya yarar. E-ticaret, blog, kurumsal ya da kişisel fark etmeksizin tüm web sitelerin temel iskeletini bu işaretleme dili oluşturur
